# Thomas Percival - Consulting Portfolio

A lightweight, high-performance landing page for my automation consulting 
services. Built to highlight the £100 custom tool offering, process, and 
contact information for prospective B2B clients.

## Tech Stack

* Framework: Next.js (App Router)
* Styling: Tailwind CSS
* Fonts: Inter & Playfair Display (Google Fonts)
* Deployment: Vercel / GitHub Pages

## Local Development

To run this project locally, clone the repository and install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Project Purpose

This frontend acts as the primary lead conversion asset for outbound marketing efforts. It is strictly scoped to replace a traditional "junior developer CV" with a "B2B consultant" layout, focusing purely on client pain points, delivery speed, and actionable results.
